## Get started in 3 steps

<Steps>
  <Step title="Deploy Kill Bill">
    Run Kill Bill on AWS with the one-click deployer, or start locally with Docker. The server exposes its REST API at `http://localhost:8080`.
  </Step>
  <Step title="Create your first tenant and catalog">
    Set up a tenant with API key and secret, then upload a product catalog defining your plans and pricing.
  </Step>
  <Step title="Create an account and subscription">
    Call the API to create a customer account, add a payment method, and start a subscription. Kill Bill automatically generates the first invoice.
  </Step>
</Steps>

<Note>
  Kill Bill is open-source under the Apache 2.0 license. Self-host on your own infrastructure or use [Kill Bill Cloud](https://cloud.killbill.io) for free hosted sandboxes and developer tools.
</Note>
